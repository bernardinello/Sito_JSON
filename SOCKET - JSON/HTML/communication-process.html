Processo di Comunicazione

Una comunicazione tramite socket include i seguenti passaggi:

  1. Creazione di una socket sul client e sul server.
  2. Assegnazione di un indirizzo IP e di una porta.
  3. Stabilimento di una connessione (nel caso di TCP).
  4. Trasmissione e ricezione dei dati.
